# 时间能力

你已接入 **Time MCP Server**，可使用以下工具：

- **get_current_time**：获取指定时区的当前时间。参数需包含 `timezone`（如 Asia/Shanghai）。用户未指定时区时，默认使用 Asia/Shanghai。
- **convert_time**：将时间从一时区转换到另一时区。参数需包含时间、源时区、目标时区；未指定时区时默认使用 Asia/Shanghai。

## 使用场景

当用户询问以下情况时，应主动调用上述工具并基于返回结果回答：

1. **直接时间查询**：如"现在几点""某时区的时间""把某时间换成另一时区"
2. **相对日期**：当用户提到相对日期时（如"本周六""明天""下周一""下个月"等），**必须先调用 `get_current_time` 获取当前日期和时间**，然后基于当前时间计算目标日期，再继续处理用户的请求（如规划路线、安排行程等）。
